🔑 GitHub Token 設定（讓「↻ 更新股價」即時抓最新）
填入 Fine-grained Personal Access Token 後，按「↻ 更新股價」會直接觸發 GitHub Actions 重新抓 Yahoo Finance，不用自己進 GitHub 點按鈕。
⚠ Token 只存在你本機 localStorage，不會上傳。
如何建 Token：GitHub → Settings → Developer settings → Personal access tokens → Fine-grained tokens → Generate → 選 BeRich repo、Actions: Read and write
☁ 跨裝置同步（GitHub Gist）
填入 Gist ID 後，每次資料變動會自動上傳到雲端，其他裝置打開時可下載最新。
需要：① 已建一個 Secret Gist（檔名 berich-data.json）② Token 的 Gists 權限為「Read and write」
💡 使用流程：① Mac 上設好資料 → ☁ 立即上傳 ② Windows 打開 → ⬇ 從雲端下載 → 之後改的會自動同步
⚠️ 注意：下載會覆蓋本地資料。若兩邊都改過，請先在「最新」那邊上傳，再到另一邊下載。
⚙ 進階：自訂 CORS Proxy（如果公用 proxy 失敗）
填入你的 Cloudflare Worker URL，會優先使用此 proxy 抓股價。留空則用公用 proxy。
範例格式：https://my-proxy.your-name.workers.dev
